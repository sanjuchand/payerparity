# PayerParity

Marketing website for PayerParity — payer-sourced negotiation leverage for hospitals.

## Website

The Next.js site lives in [`website/`](./website/).

```bash
cd website
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy on Vercel

1. Import this repository in [Vercel](https://vercel.com/new).
2. Set **Root Directory** to `website`.
3. Deploy — Vercel auto-detects Next.js.

## Blog posts

Add markdown files to `website/content/blog/` with frontmatter (`title`, `description`, `date`, `author`, `published`).
